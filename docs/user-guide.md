### AI Search
1. Navigate to Admin panel
2. Select "AI Search" tab
3. Enter query like "AI image generator"
4. Click "Search"
5. Add relevant tools to directory

## Tool Submission
1. Click "Add Tool" navigation
2. Fill out:
    - Tool name
    - Description
    - URL
    - Category
3. Submit form
4. Wait for admin approval
